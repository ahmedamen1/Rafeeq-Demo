SLIDE 3 Autonomous Navigation Stack — Sensors, Costmaps, Planner ~2 min
Purpose

Show how Nav2 takes the saved map + live sensor data and drives the robot to a goal. The key story is the two-sensor fusion: 2D LiDAR for the obstacle layer + Kinect/depth camera for the voxel layer (catches table-height obstacles the LiDAR misses). MPPI controller, NavFn global planner, AMCL localisation.

Designer Sensor fusion callout box
Gold-bordered callout box highlighting the two-sensor story: "LiDAR detects walls and floor-level obstacles · Depth camera detects raised obstacles (tables, chairs) — covering the full 0–2 m height range the LiDAR misses."
Presenter note: "Once the map is saved, Nav2 takes over. AMCL localises the robot on the map using the live LiDAR. The global planner finds a path, and the MPPI controller executes it in real time, sampling 10,000 candidate trajectories every cycle. The key design decision here is the two-layer costmap: the LiDAR handles walls and floor-level obstacles, and the depth camera covers everything above that — specifically to handle indoor furniture like tables that the 2D LiDAR scan plane completely misses."
